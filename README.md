# Flow-Free-Game
# 🎮 Flow Free Advanced Puzzle Game

A Python-based interactive puzzle game inspired by Flow Free, built using Pygame.  
The game combines algorithmic puzzle generation with basic AI concepts to create dynamic, solvable puzzles.

---

## ✨ Features

- 🔄 Dynamic puzzle generation using DFS & Backtracking  
- 🎯 Interactive grid-based gameplay  
- 🎮 Multiple colored path connections  
- 🚫 No overlapping or intersecting paths allowed  
- 🧠 Automatic win detection  
- 🔁 Reset game functionality (Press R)  
- 🖱️ Mouse-based intuitive controls  

---

## 🛠️ Technologies Used

- Python  
- Pygame  
- Algorithms (DFS, Backtracking)  
- Basic AI Concepts  

---

## 🎮 How to Play

1. Click on a colored dot to start a path  
2. Drag the mouse to connect the path  
3. Connect each color’s two endpoints  
4. Fill the entire grid without overlapping paths  
5. Win when all conditions are satisfied 🎉  

---

## ⌨️ Controls

- Mouse Click & Drag → Draw paths  
- R Key → Reset game  
- Close Window → Exit game
